# Excel to json converter

## How to use

Just rename the excel file to `demo.xlsx` and put it in root folder.
#### Important ! 
###### All unused excel sheets (sheets without data) need to be deleted!

- `npm install`
- `node index.js`
- see result in `result.json` (in root folder)
