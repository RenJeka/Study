#### _Desined by using AWS services:_
* Lambda function
* DynamoDB
* S3 bucket Storage
* API GateWay
* IAM


#### **Tech stack:**
* Vinilia JS
* NodeJs 12
* pico css framework (https://picocss.com/)


## What You can do with App?:
- Add you favorite task for todo;
- Delete your todo task, if it has been done;
